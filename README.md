# Awesome Solana Zig [![Awesome](https://awesome.re/badge.svg)](https://github.com/sindresorhus/awesome)

> List of all repos that are related to development in **zig** on **solana**, directly or not, everything we in ziglana devving ever used will be here

## Infra

- [Sig](https://github.com/Syndica/sig) - a Solana validator client implementation written in Zig 

## Program Development

- [solana-program-library-zig](https://github.com/joncinque/solana-program-library-zig) - Zig package for interacting with SPL programs 
- [solana-program-sdk-zig](https://github.com/joncinque/solana-program-sdk-zig) - Write Solana programs in Zig.
